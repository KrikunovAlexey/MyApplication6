В игре кликните "~" и набирайте доступные читы.
Или кликните комбинацию "Ctrl + Alt + ~".
"aviDemo" для записи вашей демки в avi файл,
"benchmark" для теста скорости игры,
"doomhell" для перехода на последний уровень,
"freeze" остановить всех,
"gfxinfo" для вывода инфы по граф. карте,
"god" для бессмертия,
"noclip" для прохождения через стены,
"notarget" для невидимости противников,
"com_drawfps 1" для вывода FPS,
"give all" для полного боекомплекта,
"give doom95" играть в старый дум,
"give keys" для всех ключей,
"give weapon_machinegun" для Machinegun,
"give weapon_shotgun" для двустволки,
"give weapon_plasmagun" для плазменной пушки,
"give weapon_bfg" для BFG,
"give weapon_chainsaw" для бензопилы,
"give weapon_rocketlauncher" для ракетницы,
"status" вывести статус игры,
"map <имя>" открыть карту,
"spawn <имя>" получить предмет,
"quit" выход.

Чтобы сразу стартовать с тяжёлого уровня, отыщем файл "doomconfig.cfg" и поменяйте в нем строчку
seta g_nightmare "0" на seta g_nightmare "1"
Настройки игры
r_gamma 3 <- прибавить к гамме
r_brightness 0
seta g_showPlayerShadow "0" <- убрать тeни персонажа,
seta g_showPlayerShadow "1" <- добавить тeни персонажа.
